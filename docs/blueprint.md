# **App Name**: StudyVerse

## Core Features:

- User Authentication: Secure login, registration, and profile management.
- Course and Note Pages: Create and manage course materials and notes with distinct page types.
- Organized Notebook System: Implement a multi-level notebook system with binders, notebooks (with colors and tags), and individual pages for courses and notes.
- AI-Powered Speech-to-Text: Real-time speech-to-text functionality for note-taking.
- AI-Enhanced Note Export: Refine and organize notes into well-structured course documents, utilizing AI tool.
- AI Diagram Generation: Generate diagrams in TXT + ASCII format from textual information, and also make the diagram compatible with markown and latex utilizing AI tool.
- Collaborative Document Sharing: Enable users to share files with each other within the platform.
- Google Workspace Theme: Adopts the Google Workspace theme.

## Style Guidelines:

- Primary color: Slate blue (#708090) to create a calm and intellectual learning environment.
- Background color: Very light grey (#F5F5F5), for a clean and unobtrusive backdrop.
- Accent color: Light orange (#F08080) for interactive elements.
- Body and headline font: 'Alegreya', a contemporary serif.
- Use minimalist, academic-themed icons for navigation and key features.
- Clean, well-spaced layout with clear visual hierarchy to enhance readability and focus.
- Use very simple animations during speech-to-text loading.
